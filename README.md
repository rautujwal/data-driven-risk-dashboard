# Data-Driven Risk Dashboard

**Business Risk Quantification | Predictive Analytics | Visualizations | AI Recommendations**

A professional dashboard that helps businesses **quantify pricing risks, predict outcomes, and make informed decisions** using machine learning and interactive visualizations.

## Highlights

- **Predictive Modeling:** Uses **Random Forest Regressor** to forecast pricing outcomes.  
- **High Accuracy:** Achieves **RÂ² score of 0.95** on test data.  
- **Risk Assessment:** Identifies high-risk scenarios that may impact revenue or customer churn.  
- **What-If Simulations:** Dynamically evaluate business decisions and their impact.
- **Visualizations:** Clear charts and plots display prediction results, risk distribution, and scenario analysis.
- **AI Recommendations:** Provides actionable insights to optimize outcomes.  


## Tech Stack

- Python, Pandas, NumPy  
- Scikit-learn (Random Forest Regressor)  
- Flask API backend  
- HTML/CSS frontend for dashboard  
- Pickle files for model and preprocessed data
